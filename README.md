# PlaywrightLearning

Playwright learning exercises and examples.

Project structure:

- Chapter_01_Basics/: Basic Playwright scripts and setup verification

Getting started:

1. Install dependencies (if you use npm):

```bash
npm install
npx playwright install
```

2. Run tests/scripts (example):

```bash
npx playwright test
```

Notes:

- This repo contains example scripts; add a `package.json` and Playwright config to run tests.
